دستیار هوشمند گفتگو، ثبت عوارض دارویی و درخواست مشاوره، کاملاً سازگار با المنتور و دارای پنل مدیریت اختصاصی.

== Description ==

افزونه «دستیار هوشمند نفس» یک ویجت چت شناور حرفه‌ای برای سایت‌های وردپرسی فارسی (RTL) است که عیناً بر اساس پورتال آموزش بیمار نفس فارمد بازسازی شده است.

**ویژگی‌ها:**

* دکمه شناور و پنجره چت با ظاهر مدرن و واکنش‌گرا (Responsive).
* منوی اصلی با ۴ بخش: سوال درباره شرکت، سوال درباره محصولات، ثبت عوارض دارویی، درخواست مشاوره.
* گفتگوی هوشمند با پشتیبانی از Google Gemini یا Webhook سفارشی (با پیام جایگزین در حالت غیرفعال).
* فرم ثبت عوارض و درخواست مشاوره با اعتبارسنجی کامل.
* ذخیره درخواست‌ها در دیتابیس وردپرس + خروجی CSV.
* اعلان لحظه‌ای به بات بله / تلگرام و ایمیل.
* ویجت اختصاصی المنتور با کنترل‌های سفارشی‌سازی بالا (متن‌ها، رنگ‌ها، موقعیت، محصولات، گزینه‌های منو).
* پنل مدیریت کامل در وردپرس برای تنظیمات و مدیریت درخواست‌ها.
* پشتیبانی از تم روشن / تیره / خودکار.
* امنیت کامل: nonce، sanitize، محدودیت نرخ درخواست.

== Installation ==

1. پوشه `nafas-chatbot` را در مسیر `/wp-content/plugins/` آپلود کنید (یا فایل zip را از منوی افزونه‌ها نصب کنید).
2. افزونه را از منوی «افزونه‌ها» فعال کنید.
3. به منوی «دستیار هوشمند» در پیشخوان بروید و تنظیمات را انجام دهید.
4. برای نمایش خودکار، گزینه «دکمه شناور» را فعال کنید؛ یا از ویجت المنتور / شورت‌کد `[nafas_chatbot]` استفاده کنید.
