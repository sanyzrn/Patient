## 📖 درباره پروژه

این پروژه یک **اپلیکیشن تک‌صفحه‌ای (SPA)** واکنش‌گرا و کاملاً راست‌به‌چپ است که برای آموزش و حمایت از بیماران شرکت دانش‌بنیان **نفس زیست فارمد** ساخته شده است. کاربران می‌توانند کاتالوگ‌های دارویی را مانند یک کتاب واقعی ورق بزنند، روی صفحات یادداشت بگذارند، ویدئوهای آموزشی ببینند و از طریق دستیار هوشمند سوال بپرسند، عوارض دارویی را گزارش کنند یا درخواست مشاوره ثبت کنند.

اپلیکیشن به‌صورت **PWA** ساخته شده و قابلیت نصب روی موبایل و استفاده آفلاین را دارد.

## ✨ امکانات کلیدی

### 📚 نمایشگر کتاب دیجیتال (Flipbook)
- ورق‌زدن سه‌بعدی و واقع‌گرایانه با افکت صوتی
- پشتیبانی همزمان از **فایل‌های PDF** (رندر زنده با `pdf.js`) و **صفحات تصویری**
- ابزار **هایلایت** و **یادداشت‌گذاری** روی صفحات (ذخیره محلی)
- **زوم و جابجایی** (Pinch-to-zoom)، حالت تمام‌صفحه، نمایش تک‌صفحه/دوصفحه
- **فهرست مطالب (TOC)** برای پرش سریع، نوار تصاویر بندانگشتی با اسکرول خودکار
- **ذخیره آفلاین**، دانلود و اشتراک‌گذاری لینک مستقیم صفحه
- ناوبری کامل با کیبورد و دسترس‌پذیری (a11y)

### 🎬 رسانه و محتوا
- گالری **ویدئوهای آموزشی** با پلیر اختصاصی
- **اسلایدر بنر** هوشمند (متنی/تصویری) با توقف هوشمند و پشتیبانی swipe
- **جستجوی زنده**، فیلتر دسته‌بندی، مرتب‌سازی و نمایش شبکه‌ای/لیستی

### 🤖 دستیار هوشمند (چت‌بات)
- منوی چندبخشی: سوال درباره **شرکت** و **محصولات**، **ثبت گزارش عوارض دارویی (ADR)** و **درخواست مشاوره**
- ارسال اعلان لحظه‌ای فرم‌ها به **بات بله** و ذخیره در سرور
- > 💡 این چت‌بات به‌صورت یک **افزونه مستقل وردپرس + المنتور** نیز توسعه داده شده است. [مشاهده مستندات افزونه ↗](./wordpress-plugin/README-FA.md)

### 🎨 تجربه کاربری
- سه پوسته **روشن / تیره / مطالعه (Sepia)** با ماندگاری و تبعیت از تنظیمات سیستم
- انیمیشن‌های روان با `Motion`، طراحی موبایل‌محور، فونت **وزیرمتن**
- دکمه بازگشت به بالا، شمارنده نتایج و حالت‌های خالی هوشمند

### 🛠️ پنل مدیریت
- دسترسی محافظت‌شده با رمز (میانبر `Alt + A` یا `#admin`)
- مدیریت کامل **کاتالوگ‌ها، ویدئوها و بنرها** (افزودن/ویرایش/حذف، آپلود تصویر)
- داشبورد **آمار و آنالیز**: نمودار میزان بازدید کاتالوگ‌ها و زمان توقف کاربر روی هر صفحه
- مدیریت مستقیم دیتابیس از طریق ویرایش/کپی کد `data.js`

## 🚀 راه‌اندازی

> **پیش‌نیاز:** [Node.js](https://nodejs.org/) نسخه ۱۸ یا بالاتر

```bash
# ۱) نصب وابستگی‌ها
npm install

# ۲) اجرای محیط توسعه (http://localhost:5173)
npm run dev

# ۳) ساخت نسخه تولید در پوشه dist/
npm run build

# ۴) پیش‌نمایش نسخه ساخته‌شده
npm run preview
```

## 🗄️ مدیریت داده‌ها

داده‌ها با اولویت زیر بارگذاری می‌شوند:

1. **فایل خارجی** `public/data.js` (متغیر `window.NAFAS_DATA`) — منبع اصلی در محیط تولید
2. **LocalStorage** — تغییراتی که از طریق پنل مدیریت ذخیره می‌شوند
3. **`initialData.ts`** — داده‌های پیش‌فرض داخلی

برای بروزرسانی محتوای سایت کافی است از پنل مدیریت کد جدید را کپی کرده و در فایل `public/data.js` جایگذاری کنید.

## 🌍 استقرار (Deployment)

پروژه با `base: './'` ساخته می‌شود و قابل میزبانی در هر مسیر/زیرپوشه است.

1. خروجی `npm run build` (پوشه `dist/`) را روی هاست آپلود کنید.
2. فایل‌های `chat.php` و `submit_form.php` را روی یک **هاست با پشتیبانی PHP** قرار دهید (برای عملکرد چت‌بات و فرم‌ها).
3. توکن و شناسه چت **بات بله** را داخل `submit_form.php` تنظیم کنید.

> چون اپلیکیشن PWA است، پس از اولین بازدید، محتوا برای استفاده آفلاین کش می‌شود.
